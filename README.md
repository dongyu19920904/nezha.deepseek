## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/yourusername/deepseek-tutorial.git
```

2. 打开项目
```bash
cd deepseek-tutorial
```

3. 在浏览器中打开 `index.html` 或使用本地服务器
