🔥 三昧真火提示词模板

（混天绫卷出三根金签）

签1：信息穿串模板

帮我的[手机/奶茶店/新衣服]写卖点：
1. 先找出3个最牛的特点（像找烤翅最嫩的部位）
2. 用小学生都懂的话说（比如'充电快得像窜天猴'）
3. 加个对比参照物（'屏幕比脸盆还大'）

【示例】帮我的智能手机写卖点：

  1. 充电5分钟刷剧2小时（像给手机灌了红牛）
  2. 拍照自带美颜不用P（比相亲照还真实）
  3. 摔地上三次都不碎（比鸡蛋壳结实）

签2：情感点火模板

帮我的[电动车/棉被/书包]写走心文案：
1. 找使用者最常待的地方（工地/学校/夜市）
2. 写三种感官体验（闻到的/听到的/摸到的）
3. 联系TA最在意的人（孩子/父母/对象）

【示例】帮我的保暖内衣写文案：

"凌晨送外卖时（👃闻见路边摊的葱花香）
（👂听见手机接单的叮咚声）
（✋摸着冻僵的膝盖）
想起媳妇织的毛裤还没舍得穿..."

签3：行动召唤模板

帮我的[早餐店/理发店/水果摊]搞促销：
1. 设个倒计时优惠（前XX名送卤蛋）
2. 操作要比划拳还简单（扫码就行）
3. 吓唬人会错过好事（像错过末班车）

👥 顾客画像与传播方案（三眼锁定法） 

帮我为[产品/服务]设计一个精准的顾客画像和传播方案，要求：

1. 顾客画像：
   - 画出3种典型顾客的详细形象
   - 每种顾客都要包括：年龄、职业、收入、兴趣爱好、消费习惯
   - 给每种顾客起个生活化的昵称（比如"夜宵王小明"）

2. 传播渠道：
   - 针对每种顾客，设计最适合的3个宣传渠道
   - 说明为什么这些渠道最适合他们
   - 每个渠道的具体投放时间和频率

3. 互动彩蛋：
   - 设计3个能让顾客主动参与的小活动
   - 每个活动都要简单易懂，一看就会
   - 说明参与后能得到什么奖励

4. 效果预估：
   - 预计每种顾客的转化率大概是多少
   - 哪种顾客最值得重点投入

【实例】帮我为"手工皮具店"设计一个精准的顾客画像和传播方案

包含顾客画像、传播渠道、互动彩蛋和效果预估四个部分，针对手工皮具店的目标客户群体进行精准定位和营销策略设计。

📅 活动执行计划书（渡劫清单法） 

帮我为[活动名称]制定一个详细的执行计划，要求：

1. 时间表（倒推法）：
   - 活动当天是[日期]
   - 从现在到活动当天，每周需要完成什么任务
   - 重点标出3个最关键的时间节点

2. 人员分工表：
   - 列出活动需要的所有角色（至少5个）
   - 每个角色负责什么具体任务
   - 谁来监督检查每个任务的完成情况

3. 应急预案：
   - 列出可能出现的5个突发问题
   - 每个问题的解决方案
   - 谁负责处理这些突发情况

4. 物资清单：
   - 活动需要准备的所有物品
   - 每种物品的数量和预算
   - 采购时间和负责人

5. 成功标准：
   - 如何判断这个活动是否成功
   - 设定3个可量化的目标

【实例】帮我为"社区亲子烘焙大赛"制定一个详细的执行计划

包含时间表、人员分工表、应急预案、物资清单和成功标准五个部分，为社区亲子烘焙大赛提供全面的活动执行规划。

💎 价值主张设计（三重境界法） 

帮我为[产品/服务]设计一个有力的价值主张，要求：

1. 功能价值（实际好处）：
   - 列出[产品/服务]能解决的3个具体问题
   - 每个问题的解决方案要具体明确（比如"1分钟内完成"而不是"快速完成"）
   - 用数字或具体事实说明效果（比如"省30%时间"而不是"省很多时间"）

2. 情感价值（感受体验）：
   - 描述使用[产品/服务]后的情感体验
   - 这种体验如何与顾客的生活产生共鸣
   - 用一个生活中的比喻来形容这种感受

3. 差异化价值（与众不同）：
   - 与市场上3个主要竞争对手相比有什么不同
   - 为什么顾客应该选择我而不是竞争对手
   - 这种差异对顾客的实际意义

4. 价值表达方式：
   - 设计3个简短有力的价值宣言
   - 每个宣言都要具体而非空泛
   - 避免使用行业术语，用普通人能理解的语言

【实例】帮我为"老奶奶手工编织围巾"设计一个有力的价值主张

包含功能价值、情感价值、差异化价值和价值表达方式四个部分，为老奶奶手工编织围巾打造独特的市场定位和价值主张。

⚡ 品牌发展规划（混天绫展宏图） 

帮我为[品牌/店铺]制定一个5年发展规划，要求：

1. 分阶段目标设定：
   - 第1年：设定3个具体可衡量的目标
   - 第3年：设定3个中期发展目标
   - 第5年：设定3个长期愿景目标
   - 每个目标都要有具体数字或明确标准

2. 实现路径规划：
   - 每个阶段需要做哪些具体工作
   - 需要投入哪些资源（人力/资金/技术）
   - 可能遇到的主要障碍和解决方案

3. 品牌升级计划：
   - 产品/服务如何逐步升级
   - 目标客户群如何扩展
   - 品牌形象如何提升

4. 避免的陷阱：
   - 如何避免"画大饼"不落地
   - 如何确保每个阶段目标可执行
   - 如何设定合理的期望值

【实例】帮我为"小明水果店"制定一个5年发展规划

包含分阶段目标设定、实现路径规划、品牌升级计划和避免的陷阱四个部分，为小明水果店规划长期可持续发展路径。

📱 小店年度总结（风火轮法） 

帮我写一份[小店/摊位]的年度经营总结，要求：

1. 销售业绩回顾：
   - 用"数钱法"列出今年最畅销的3种产品/服务及其销售数据
   - 用"说书法"讲述一次成功的促销活动或解决顾客难题的故事
   - 用"证据法"提供3个证明店铺进步的数据（比如客流量增长、好评率等）

2. 店铺发展亮点：
   - 我们团队完成的2-3项改进（比如装修、新品开发等）
   - 我们创新的1-2个经营方法或服务流程
   - 我们获得的1-2个认可或荣誉（比如社区推荐、平台好评等）

3. 来年经营规划：
   - 短期目标（1-3个月）：列出2-3个具体可执行的改进计划
   - 中期目标（半年）：列出2-3个店铺升级计划
   - 长期目标（一年）：描述店铺的愿景和发展方向

4. 感谢与展望：
   - 感谢2-3类关键支持者（比如老顾客、合作伙伴等）
   - 对未来的期许和承诺
   - 一句温暖有力的结束语

【实例】帮我写一份"老王烧烤摊"的年度经营总结

包含销售业绩回顾、店铺发展亮点、来年经营规划和感谢与展望四个部分，全面总结老王烧烤摊一年来的经营成果和未来规划。

框架方法

📝 公众号文章结构设计（乾坤圈法） 

帮我设计一篇关于[主题]的公众号文章结构，要求：

1. 标题设计（3选1）：
   - 设计3个不同类型的标题：
     * 问题型标题（引发思考）
     * 数字型标题（提供明确价值）
     * 故事型标题（激发好奇心）

2. 开篇设计（钩子）：
   - 设计3种不同的开场方式：
     * 提出读者痛点
     * 讲述一个简短故事
     * 抛出一个有争议的观点

3. 内容框架（主体）：
   - 设计一个清晰的内容结构，包括：
     * 3-5个主要部分
     * 每部分的小标题
     * 每部分应该包含的要点
     * 在哪些部分插入案例或数据

4. 结尾设计（收尾）：
   - 设计2种不同的结尾方式：
     * 总结型结尾（强化主要观点）
     * 行动型结尾（鼓励读者采取行动）

5. 互动设计：
   - 在文章中设置2-3个互动点，如：
     * 提问引导评论
     * 投票或选择题
     * 邀请分享经验

【实例】帮我设计一篇关于"如何培养孩子的阅读习惯"的公众号文章结构

包含标题设计、开篇设计、内容框架、结尾设计和互动设计五个部分，为公众号文章提供完整的结构规划。

📝 微博内容创作（火尖枪法） 

帮我创作一条关于[主题]的微博内容，要求：

1. 内容结构（三段式）：
   - 开头：用一句话抓住读者注意力
   - 中间：用2-3个简短段落展开主题
   - 结尾：用一句话总结或引发思考

2. 表达风格（选择一种）：
   - 幽默调侃型
   - 干货分享型
   - 情感共鸣型
   - 观点评论型

3. 互动元素（至少包含2个）：
   - 设计一个简短有趣的话题标签
   - 加入1-2个与内容相关的表情
   - 设计一个引发评论的问题
   - 添加一个投票或二选一问题

4. 视觉元素建议：
   - 描述应该配什么类型的图片
   - 如果需要，提供简单的图片文字说明

5. 转发策略：
   - 设计一句鼓励转发的话
   - 提供1-2个转发理由

【实例】帮我创作一条关于"夏日防晒小技巧"的微博内容

包含内容结构、表达风格、互动元素、视觉元素建议和转发策略五个部分，为微博内容提供全面的创作指导。

🧠 小红书人设打造（混天绫法） 

帮我设计一个[领域]的小红书博主人设，要求：

1. 人设基础（形象定位）：
   - 设计一个独特的人设名称（比如"都市野餐规划师"而非简单的"美食博主"）
   - 3-5个关键人设标签（比如"85后、理工男、烘焙达人、猫奴、露营爱好者"）
   - 一句简短有力的个人简介
   - 头像和背景图的风格建议

2. 内容规划（内容矩阵）：
   - 设计3-5个内容主题方向
   - 每个主题下的3个具体选题示例
   - 建议的更新频率
   - 内容风格和语言特点

3. 互动策略（社区运营）：
   - 如何回复粉丝评论的风格和原则
   - 3个能引发高互动的提问或话题
   - 如何与其他博主互动和联动
   - 如何处理负面评论

4. 差异化亮点（脱颖而出）：
   - 与同领域博主的3个主要区别
   - 能体现专业性的2-3个细节
   - 能增加亲和力的2-3个设定

【实例】帮我设计一个"家居收纳"的小红书博主人设

包含人设基础、内容规划、互动策略和差异化亮点四个部分，为小红书博主打造独特且有吸引力的人设形象。

🎬 抖音开场设计（风火轮法） 

帮我设计一个[主题]的抖音视频开场，要求：

1. 开场钩子（3秒吸引力）：
   - 设计3种不同类型的开场白：
     * 悬念型（制造好奇）
     * 冲突型（设置矛盾）
     * 福利型（直接给价值）
   - 每种开场白不超过15个字
   - 语气要口语化，像跟朋友聊天

2. 画面设计（视觉冲击）：
   - 第一个镜头应该展示什么
   - 有什么视觉效果或道具
   - 表情和肢体动作建议

3. 节奏设计（韵律感）：
   - 开场语速应该快还是慢
   - 在哪个词需要停顿或强调
   - 配乐或音效建议

4. 过渡设计（承上启下）：
   - 如何从开场自然过渡到主体内容
   - 一句承接的过渡语
   - 过渡时的画面变化建议

【实例】帮我设计一个"厨房小技巧"的抖音视频开场

包含开场钩子、画面设计、节奏设计和过渡设计四个部分，为抖音视频提供吸引人的开场方案。

🍲 火锅点餐法（TASTE框架） 

【TASTE框架】请帮我[任务描述]，使用火锅点餐法：

T - Topic（主题）：
   - 我需要解决的问题/完成的任务是[具体描述]
   - 这个任务的背景是[背景信息]
   - 最终目标是[期望结果]

A - Audience（受众）：
   - 这个内容/方案的目标受众是[受众描述]
   - 他们的主要特点是[特点描述]
   - 他们最关心的问题是[关注点]

S - Style（风格）：
   - 我希望采用的风格是[风格描述]，例如[举例]
   - 语言应该[正式/轻松/专业/生活化]
   - 应该避免[不希望的风格元素]

T - Template（模板）：
   - 内容结构应该包括[结构描述]
   - 可以参考的成功案例是[案例]
   - 特别需要强调的部分是[重点]

E - Extra（额外要求）：
   - 特别需要注意[特殊要求]
   - 需要避免的雷区是[禁忌]
   - 其他补充说明[补充]

【实例】请帮我写一篇关于"如何选择适合自己的健身方式"的文章，使用火锅点餐法

使用TASTE框架（Topic、Audience、Style、Template、Extra）来明确文章的主题、目标受众、风格、结构模板和额外要求，为健身方式选择文章提供清晰的创作指导。

👟 球鞋购买法（ALIGN框架） 

【ALIGN框架】请帮我[任务描述]，使用球鞋购买法：

A - Aim（目标）：
   - 我的主要目标是[目标描述]
   - 成功的标准是[成功标准]
   - 这个任务的优先级是[优先级]

L - Limitations（限制）：
   - 我面临的主要限制是[限制描述]
   - 可用的资源有[资源描述]
   - 必须遵守的规则是[规则]

I - Ideas（创意）：
   - 我已经想到的方法是[已有想法]
   - 希望探索的新方向是[新方向]
   - 特别感兴趣的参考案例是[参考]

G - Ground（基础）：
   - 我已经掌握的相关知识/技能是[已有基础]
   - 我的优势是[优势]
   - 我需要补充学习的是[学习需求]

N - Next Steps（下一步）：
   - 完成这个任务后，我计划[后续计划]
   - 希望这个任务能帮助我[长期目标]
   - 我希望得到的具体建议是[具体需求]

【实例】请帮我制定一个"30天减肥计划"，使用球鞋购买法

使用ALIGN框架（Aim、Limitations、Ideas、Ground、Next Steps）来明确减肥目标、限制条件、创意方法、基础情况和后续计划，为30天减肥计划提供全面的规划指导。

知识觉醒大法

🔥 知识分级法（辣翅分级） 

【知识分级学习】请帮我学习[主题]，按照难度分级：

1. 基础知识（微辣级）：
   - 请用最简单的话解释[主题]的核心概念
   - 只使用日常生活中的例子
   - 避免任何专业术语，就像跟小学生解释一样
   - 重点解释这3个最基础的概念：[列出3个核心概念]

2. 进阶知识（中辣级）：
   - 在理解基础概念后，请解释这些稍复杂的内容
   - 可以使用一些简单的专业术语，但要立即解释
   - 用类比帮助理解这些概念
   - 重点解释这3个进阶概念：[列出3个进阶概念]

3. 高级知识（变态辣级）：
   - 只有在完全理解前两级后才学习
   - 可以使用专业术语和框架
   - 解释这些概念之间的复杂关系
   - 重点解释这2个高级概念：[列出2个高级概念]

请先从微辣级开始，我理解后再进入下一级。

【实例】学习"手机摄影"

按照微辣级（基础知识）、中辣级（进阶知识）和变态辣级（高级知识）三个难度等级，逐步学习手机摄影的核心概念、技巧和高级应用。

🍗 烧烤实操法（学习应用） 

【知识烧烤实操】请帮我学习如何应用[知识/技能]，就像烤鸡翅一样分步骤：

1. 火候控制（参数调整）：
   - 请列出学习这个技能时需要注意的3-5个关键参数
   - 每个参数的最佳"火候"是什么
   - 如何根据不同情况调整这些参数
   - 常见的"火候"掌握错误有哪些

2. 翻面时机（换角度思考）：
   - 当遇到[常见困难1]时，有什么替代方法
   - 当遇到[常见困难2]时，如何换个角度解决
   - 提供3个不同视角来理解这个知识点
   - 什么情况下应该暂停并重新思考方向

3. 秘制酱料（经验总结）：
   - 请提供一个简单易记的"配方"来记住关键步骤
   - 设计一个口诀或记忆法帮助记忆重点
   - 分享3个提高效率的小技巧
   - 如何把这个经验应用到其他相关领域

请用生活化的语言和具体例子解释，避免抽象概念。

【实例】学习"做简易家常菜"

通过火候控制（参数调整）、翻面时机（换角度思考）和秘制酱料（经验总结）三个步骤，学习如何应用做简易家常菜的技能。

⚡ 知识觉醒三连招（创新学习法） 

【知识觉醒三连招】请帮我用创新方式学习[主题]：

1. 回忆高光时刻（找到相似经验）：
   - 请问我之前是否有过类似的学习经验：[描述你可能有的相关经验]
   - 在那次经验中，我成功掌握了什么技能
   - 当时是用了什么方法学会的
   - 那次经验中最关键的突破点是什么

2. 连接新招式（建立知识联系）：
   - 请帮我找出[旧知识点]和[新知识点]之间的3个相似之处
   - 解释如何把旧知识的学习方法应用到新知识上
   - 设计一个练习，把两种知识结合起来
   - 找出可能阻碍知识连接的"思维盲点"

3. 创造必杀技（知识创新应用）：
   - 基于前两步，创造一个独特的学习方法或记忆技巧
   - 设计一个只有10分钟的"闪电学习"计划
   - 提出一个创新的应用场景
   - 给这个"必杀技"起个容易记住的名字

请用生动有趣的语言回答，就像在教一个充满好奇心的孩子。

【实例】学习"英语口语"

通过回忆高光时刻（找到相似经验）、连接新招式（建立知识联系）和创造必杀技（知识创新应用）三个步骤，用创新方式学习英语口语。

📝 黑眼圈修炼手册（学习计划） 

【黑眼圈修炼手册】请帮我制定学习[主题]的计划：

1. 每日目标设定：
   - 设计5个小而可行的每日学习任务
   - 每个任务预计花费时间
   - 完成标准是什么
   - 如何检验学习效果

2. 定期升级计划：
   - 设计一个每周知识整理的方法
   - 每月应该如何复习和巩固
   - 什么时候可以进入下一个难度等级
   - 如何判断是否真正掌握了知识

3. 独门秘籍设计：
   - 创建一个简单的知识点标记系统
   - 设计一个进度追踪表
   - 提供3个克服学习瓶颈的方法
   - 设计一个奖励机制保持学习动力

请把计划设计得像游戏闯关一样有趣，避免枯燥的学习方式。

【实例】学习"基础编程"

通过每日目标设定、定期升级计划和独门秘籍设计三个部分，为基础编程学习制定一个游戏化的学习计划。

使用技巧

提示词优化技巧

  • 具体胜于抽象 - 使用具体的例子和数字，而不是模糊的描述
  • 分步指导 - 将复杂任务拆分为清晰的步骤，逐一引导AI完成
  • 角色设定 - 给AI设定一个特定角色，如"作为一名经验丰富的营销专家"
  • 输出格式 - 明确指定你希望的输出格式，如表格、列表或分段文本

常见陷阱与解决方案

问题：回答太笼统

解决：要求提供具体例子和数据支持，指定"至少3个具体例子"

问题：回答太专业难懂

解决：明确要求"用简单语言解释，像对10岁孩子讲解一样"

问题：回答偏离主题

解决：使用结构化提示，明确每部分需要回答的具体问题

问题：回答缺乏创意

解决：明确要求"提供3个创新的、非常规的解决方案"

进阶使用技巧

多轮对话策略

  1. 第一轮：提出基本问题，获取初步回答
  2. 第二轮：针对初步回答提出深入问题
  3. 第三轮：要求AI整合前两轮内容，形成完整方案
  4. 第四轮：请求AI对方案进行自我评估和改进
